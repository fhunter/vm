Виртуальная машина и память.
Память делится на блоки по 512 слов каждый.
Для отслеживания используется битовая карта.

На каждую виртуальную машину приходится:
Стеки: 16 слов, x2
Карта занятой памяти: 16 байт битовой карты занятых страниц (65536/512/8)
Текущий PC - 1 слово
Указатели стеков - 2x 1 байт

Итого - минимум - 16*2*2+16+1*2+2*1=100 байт
+ Требуемый объём памяти для эмуляции (кратно 512) но не более 65536 слов на машину

Файл состояния виртуальной машины:
1. Дисковая память дешёвая.
Структура состояния виртуальной машины.
Карта занятости: 16 байт. Добиваем 0 до сектора.
Сектора памяти - 128 штук.
